Installation

Please follow below instructions:

  1. Go to door_knock_delivery_17072020 folder
  2. Copy source_code folder and upload it in your server
  3. Open this file in your editor application/config/config.php
  4. Set the base_url like below:
    • $config['base_url'] = 'http://your-domain.com/';
  5. Create a database named door_knock
  6. Go to database folder
  7. Import door_knock.sql file into your database
  8. Open this file in your editor application/config/database.php
  9. Set the value in below three variables:
    • $db['default']['username'] = 'root';
    • $db['default']['password'] = '';
    • $db['default']['database'] = 'door_knock';
  10. And finally run the project by accessing http://your-domain.com

Login

If you already run the software, you supposed to be directed to the order page. You will see an admin login link at the footer of the page. Click on that and please login using default login credentials. Email: admin@doorsoft.co Password: 123456

No worry please, you can change your credentials immediate after login.

Change Password

After login, please go to Change Password from My Profile menu and change the default 123456 password by your own. You can also update this any time you need.

Change Email

Please go to Change Profile from My Profile menu and change the default admin@doorsoft.co email by your own. You can also update this any time you need.

Transfer License

You may want to setup this in your local first, test and upload all data then transfer it to hosting server. To do this, please follow below instruction:

  • In Your browser please hit this URL: http://your-installation.com/Authentication/Uninstall may be http://localhost/Authentication/Uninstall if you are transferring from local to web server. System will ask your purchase code and username, provide your username and purchase code
  • System will do the uninstall
  • Now you can install in web server
  • Then go to phpmyadmin and export your database and drop all of your tables from web phpmyadmin
  • Then import the local exported db to web phpmyadmin
  • Then go to your local installation
  • Copy images and slide images folder, zip these, upload to web installation, remove from web installation, and unzip these
  • You are done
  • Now you can access your web installation
  • Do the same procedure if you want to move the installation to another server
